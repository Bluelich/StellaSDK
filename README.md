Installation
============

Stella SDK first time installers:

    $ cd /opt
    $ git clone https://github.com/morningtec/StellaSDK.git
    $ /opt/StellaSDK/bin/stella-config --install

Updating Stella SDK to r2.0

    $ cd /opt/StellaSDK
    $ git reset --hard r2.0
    $ /opt/StellaSDK/bin/stella-config --install
